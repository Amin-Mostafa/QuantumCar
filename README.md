Quantum Car Factory

This project is a simple OOP simulation for a car factory system.

It includes different types of engines (Gas, Electric, Hybrid) and a car that can change its engine dynamically.

Main features:
- Start and stop the car
- Accelerate and brake with speed control
- Replace engine at runtime
- Hybrid engine switches automatically between gas and electric based on speed

How to run:
- Open the project in any Java IDE (IntelliJ / Eclipse)
- Run the Main class

The project was built to practice OOP concepts like:
- Encapsulation
- Polymorphism
- Abstraction
- Factory Design Pattern
